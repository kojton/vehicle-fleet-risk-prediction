# vehicle-fleet-risk-prediction
Predicting maintenance incidents and repair costs for vehicle fleets using statistical and machine learning models 
Modelowanie ryzyka incydentów i kosztów napraw w zarządzaniu flotą pojazdów

How to run: 
1. Clone the repo  
2. Install dependencies: `pip install -r requirements.txt`
3. Run `python src/generate_synthetic_fleet_data.py` to generate dataset  
4. Open `notebooks/01_EDA_fleet_incidents.ipynb` and follow the analysis
