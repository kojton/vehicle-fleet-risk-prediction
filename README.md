# Vehicle Fleet Risk Prediction

Predicting maintenance incidents and repair costs for vehicle fleets using statistical and machine learning models.  
**Modelowanie ryzyka incydentów i kosztów napraw w zarządzaniu flotą pojazdów.**

## Cel projektu

Celem projektu jest stworzenie modeli predykcyjnych, które:

- przewidują **prawdopodobieństwo oraz częstość incydentów** (awarii, kolizji, usterek),
- szacują **koszt napraw**,
- obliczają **oczekiwany koszt ryzyka** dla każdego pojazdu.

Projekt wykorzystuje dane syntetyczne odzwierciedlające realistyczne cechy pojazdów, a zastosowane metody obejmują zarówno modele statystyczne, jak i machine learning.
